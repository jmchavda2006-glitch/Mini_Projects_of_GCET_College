## 📋 Project Summaries

### 1. Student Management System (Data Structures)

* **Objective:** Master native hardware memory maps and explicit data structures.
* **Implementation:** Built strictly around raw heap storage using manual allocation nodes. Handles head pointer restructuring for safe array insertions, target updates, and unlinking procedures without leaving dangling memory remnants.

### 2. Automated Billing Management System (Java)

* **Objective:** Design an interactive enterprise utility following standard OOP paradigms.
* **Implementation:** Features administrative credential protection, stock tracking controls, real-time calculation matrices (applying automated discounts and **18%** GST rules), and handles file streams to generate clean transactional text invoice assets.

### 3. 2D Retro Space Shooting Game (Computer Graphics)

* **Objective:** Explore double-buffered real-time graphics pipelines and physics matrices.
* **Implementation:** Implements interactive keyboard loops, dynamic collision fields ($O(N)$ bounding box evaluations via coordinate absolute differences), and particle trail renderers locked inside a strict **16ms** refresh routine to hit a smooth 60 FPS output.

### 4. Automated Student ID Card Generating System (Web & Python)

* **Objective:** Automate institutional asset generation through a clean web pipeline.
* **Implementation:** A functional web form that captures student metadata and profile photographs, runs dynamic coordinate positioning algorithms using Python image processing libraries, and yields instant printable card files.

### 5. Optimal Library Book Arrangement System (DAA)

* **Objective:** Minimize book retrieval times using algorithmic restructuring.
* **Implementation:** Leverages the **Optimal Binary Search Tree (OBST)** concept via Dynamic Programming. It tracks real-time book borrowing/returning, calculates access frequencies, and reorganizes the dataset topology to guarantee minimal search costs compared to a basic sorting heuristic.
